# Section - A Gutenberg block that allows you to add other blocks inside.

## Features

- Background Image
 	- Fixed background
 	- Focal point picker
- Background Color
- Background Color Overlay
- Text Color
- Background Video
- Spacing options
- HTML Tag options

## Requirements

Latest version of [Gutenberg](https://github.com/WordPress/gutenberg) plugin needs to be installed.
